# Trackie

Trackie es una aplicación móvil para llevar el control de hábitos personales como peso, calorías y progreso físico de forma simple y visual.

## Características

* Registro de comidas y calorías
* Seguimiento de peso
* Visualización de progreso con gráficos
* Interfaz limpia y fácil de usar

## Tecnologías

* React Native (frontend)
* NestJS (backend)
* PostgreSQL (Supabase)
* Docker (contenedorización)
* Railway (deploy)

## Notas

Proyecto personal desarrollado con fines de aprendizaje y portafolio.
